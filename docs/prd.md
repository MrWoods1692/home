- 点击头像触发反转动画
- 动画完成后显示另一个头像
- 支持反复切换

### 4.4 主题切换逻辑
- 用户选择主题后立即应用到当前页面
- 主题偏好保存到本地存储
- 页面刷新或重新访问时自动应用已保存的主题

### 4.5 语言切换逻辑
- 用户选择语言后所有文本内容立即更新
- 语言偏好保存到本地存储
- 页面刷新或重新访问时自动应用已保存的语言
- 所有页面的导航栏、标题、正文内容同步切换

### 4.6 页面导航逻辑
- 点击导航栏链接跳转到对应页面
- 当前页面的导航项保持高亮状态
- 支持浏览器前进后退功能

### 4.7 灵动岛还原过渡动画
- 导航栏在页面切换或交互时展示灵动岛还原过渡动画效果

### 4.8 爱发电跳转逻辑

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 图片加载失败 | 显示占位图或默认图片 |
| 链接失效 | 保持链接可点击,由用户判断 |
| 收款码图片缺失 | 显示提示文字 |
| 本地存储不可用 | 使用默认主题和语言设置 |
| 系统主题检测失败 | 自动模式降级为亮色模式 |
| 爱发电链接失效 | 保持按钮可点击,由用户判断 |
